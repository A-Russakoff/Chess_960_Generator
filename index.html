Generate Chess 960 Board:


This is a demonstration of HTML5, CSS, & JavaScript. It features a non-trivial JavaScript application and utilizes some of the following features:

Basic HTML, CSS, and JavaScript Syntax.
Mode 'use strict' in JavaScript.
Linking between .html, .css, and .js files.
Elementary styling with CSS.
Basic functions in JavaScript.
Buttons and event handlers.
Use of HTML5 Canvas with javascript.
Loading an array of .jpg and .png images, waiting for the image array to load before enabling the program's draw features.(This was tricky and interesting, and I'm certain I will change the implementation in the near future.)

Some more information about the application:

In broad strokes, the application generates a new position by generating 5 random numbers.

1/4 for the Light-Squared Bishop.
1/4 for the Dark-Squared Bishop.
1/6 for the Queen.
1/5 for one of the Knights &
1/4 for the other.

The Rooks and King are then placed sequentially, first Rook, then King, then Rook, in the remaining three squares.

Positions are stored in a simple linear array. A function is required that is designed to ignore positions that have already been used as it searches for the square that corresponds to the random number used to place the piece. Once the board-position is generated, a simple 'switch case' object is used to draw the pieces on the canvas to the correct squares. The individual positions (squares) of the black and white pieces are always mirror images of each other, so once we have the board-position for one color, we get the other for free. The application simply paints the pieces over the background image (the board) in pairs with the a corresponding offset.

Currently the JavaScript application is roughly 150 lines of code, including handling all of the graphic objects and the event handlers for the buttons.
